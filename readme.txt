This package contains the source code and build files necessary to
build GTK+ for Windows 64-bit using nmake.

The source is unpacked tarballs with makefile changes and Win64 fixes
applied. The tarballs are:

libiconv-win     http://ftp.gnome.org/pub/gnome/binaries/win32/dependencies/win_iconv-tml-20080320.zip
gettext-0.17     http://ftp.gnome.org/pub/gnome/binaries/win32/dependencies/gettext-0.17.tar.gz
glib-2.14.6      http://ftp.gnome.org/pub/gnome/sources/glib/2.14/glib-2.14.6.tar.bz2
zlib-1.2.3       http://www.gzip.org/zlib/zlib-1.2.3.tar.gz
cairo-1.4.14     http://cairographics.org/releases/cairo-1.4.14.tar.gz
atk-1.22.0       http://ftp.gnome.org/pub/GNOME/sources/atk/1.22/atk-1.22.0.tar.bz2
libpng-1.2.6     http://prdownloads.sourceforge.net/libpng/libpng-1.2.26-no-config.tar.bz2?download
pango-1.18.4     http://ftp.gnome.org/pub/GNOME/sources/pango/1.18/pango-1.18.4.tar.bz2
gtk+-2.12.9      http://ftp.gnome.org/pub/GNOME/sources/gtk+/2.12/gtk+-2.12.9.tar.bz2

See the file instructions.txt in this directory for more information
on how to use the build scripts provided here.
